# Data Analysis Project EXCEL
## Project Overview
This project demonstrates advanced Excel techniques for managing and analyzing a retail inventory dataset.The goal was to transform raw data into a functional tracking system using lookup functions
and conditional logic to monitor stock level, pricing, and supplier information.
## In this project, l applied several core Excel functions essential for data analysis:

* **Advanced Lookups:** Used `XLOOKUP` and `VLOOKUP` to dynamically retrieve product (Category,Price,Location) based on Product ID
* **Conditional Logic:** Implemented nested `IF`, `AND` functions to automate status updates (flagging items for recorder based on quantity and stock status)
* **Aggregations:** Utilized `SUMIFS` and `AVERAGEIFS` to calculate total value and average pricing across different categories(Elecytronics,Clothing,Toys)
